# Cache Simulator (C)

A configurable cache simulator written in C that models direct-mapped, set-associative, and fully associative caches with FIFO and LRU replacement policies. Supports optional next-block prefetching and reports cache performance statistics.

## Features
- Direct-mapped, N-way set associative, and fully associative caches
- FIFO and LRU replacement policies
- Optional next-block prefetching
- Tracks cache hits, misses, memory reads, and writes
